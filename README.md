# Irene Comida Saudável 🌿

Uma landing page premium para um serviço de marmitas fitness e gourmet em Fortaleza. Este projeto foi desenvolvido com foco em alta performance, design sofisticado e copywriting estratégico.

## Skills & Funcionalidades

### Design Premium & Experiência do Usuário (UX)
- **Estética Sofisticada**: Paleta de cores em tons de verde profundo (oklch) e tipografia elegante (Fraunces & Inter).
- **Animações Fluidas**: Sistema de folhas flutuantes (`animate-float`) que cria uma atmosfera orgânica e premium.
- **Interatividade**: Micro-interações em botões e cards para um feedback visual refinado.
- **Responsividade Total**: Experiência otimizada para dispositivos móveis, garantindo acessibilidade em qualquer tela.

### Copywriting Estratégico
- **Inversão de Perspectiva**: Textos focados na transformação da rotina e benefícios reais para o cliente, indo além das características técnicas.
- **Tom de Voz Especialista**: Linguagem que transmite autoridade, cuidado e sofisticação.
- **CTAs Inteligentes**: Chamadas para ação estrategicamente posicionadas para maximizar a conversão via WhatsApp.

### Performance & SEO
- **SEO On-Page**: Hierarquia de cabeçalhos (H1-H3) estruturada com base na intenção de busca ("Marmitas Fitness Fortaleza").
- **Metadados Otimizados**: Títulos e descrições configurados para máxima taxa de clique (CTR) no Google.
- **Acessibilidade & Imagens**: Alt texts únicos e descritivos para todas as imagens, favorecendo o SEO e a inclusão.
- **Otimização Técnica**: Uso de `transform` e `opacity` para animações, minimizando o impacto na CPU e garantindo pontuações altas no Lighthouse/PageSpeed.

### Tecnologias Utilizadas
- **React 19** & **TypeScript**
- **TanStack Start** & **TanStack Router**
- **Tailwind CSS v4** (Modern CSS-first approach)
- **Lucide React** (Iconografia premium)
- **Vite** (Build tool de alta performance)

---

## Como rodar o projeto

1.  Clone o repositório.
2.  Instale as dependências:
    ```bash
    npm install
    ```
3.  Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```
4.  Acesse `http://localhost:3000` (ou a porta indicada pelo Vite).
